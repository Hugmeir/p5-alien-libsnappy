# NAME

Alien::libsnappy - libsnappy, with alien

# VERSION

Version 0.01

# SYNOPSIS

    use Alien::libsnappy;

    Alien::libsnappy->libs;
    Alien::libsnappy->libs_static;
    Alien::libsnappy->cflags;

# DESCRIPTION

`Alien::libsnappy` is an `Alien` interface to [libsnappy](https://github.com/google/snappy).
